a Discord Rich Presence (RPC) application that displays your current PC status.

## description

`wheremyPCis.py` is a (VERY) simple application designed to showcase your PC's current status on Discord. it integrates with Discord's RPC (Rich Presence), providing real-time information about your PC's resource usage.

**it includes:**
- **`[üé≤]` rich presence:** displays information such as CPU usage, RAM usage, and active process name on your Discord profile
- **`[üìù]` customization:** toggle the display of the active process name based on your preference
- **`[üì•]` system tray integration:** conveniently hide the application to the system tray
